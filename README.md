# FINOS (finos)
The Fintech Open Source Foundation (FINOS) is a Linux Foundation project dedicated to open source innovation in the financial services industry. It fosters collaboration between banks, fintech companies, and technology firms on projects like FDC3, Legend, and Morphir for financial technology interoperability and standards.
